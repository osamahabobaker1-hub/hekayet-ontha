# حكاية أنثى — Hekayet Ontha 🎁

A full-stack e-commerce platform for gifts and special occasions.

## 🚀 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Database ORM:** Prisma
- **State Management:** Zustand
- **Storage:** Base64 image handling

## ✨ Features

- Responsive product grid (2-col mobile / 4-col desktop)
- Full admin dashboard (product & order management)
- Dynamic search API
- Cart system with persistent state
- Checkout flow
- Base64 image storage (no broken links)

## 🛠️ Getting Started

```bash
npm install
npx prisma generate
npm run dev
```

## 📸 Built With

Next.js · TypeScript · Tailwind CSS · Prisma ORM · Zustand
